# UI/UX Rules - FisikaSeru National

## Mobile First
- Target layout: 360px, 390px, 412px, 768px, 1024px, 1440px.
- Gunakan utilitas Tailwind responsive (sm, md, lg, xl).

## Komponen
- Navbar/Footer konsisten.
- Card, badge, dan tombol menggunakan design tokens warna utama hijau.

## Stage Layout
- Canvas simulasi full width.
- Panel kontrol menggunakan card kaca (glass).
- Indicators row selalu 5 kolom.
- Tips dan data tersimpan tampilkan sesuai konteks.

## Aksesibilitas
- Fokus terlihat jelas.
- Kontras aman.
- Prefers-reduced-motion dihormati.
